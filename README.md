# Starknet Bridge Relayer

## Development

### Install
```sh
yarn
```

### Start development
#### Setup env
Duplicate the `example.env` file and rename it to `.env`

```sh
yarn start:dev
```

### Access swagger UI
```sh
http://0.0.0.0:3000/api
```

### Prometheus metrics
```sh
http://0.0.0.0:3000/metric
```
